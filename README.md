# LogiTrack API MVP — Логістична платформа для доставок

## Про проєкт
Це бекенд-додаток, призначений для управління логістичними процесами, який об'єднує диспетчерів, кур'єрів та клієнтів.

**Ідея MVP:** Створити надійний REST API для відстеження життєвого циклу замовлення (від створення до успішної доставки) з використанням чіткої рольової моделі доступу та управління статусами.

## Стек технологій
- **Backend:** TypeScript, NestJS
- **База даних:** PostgreSQL, TypeORM
- **Менеджер пакетів:** npm
- **Версіонування:** Git / GitHub

## Структура папок
- `src/` — вихідний код додатку (модулі, контролери, сервіси).
- `test/` — файли для e2e тестування.
- `docs/` — проєктна документація та звіти до лабораторних робіт.

## Інструкція з запуску
1. Скопіювати посилання на репозиторій із сайту GitHub.
2. Виконати команду `git clone [URL-адреса_репозиторію]` у терміналі.
3. Перейти у папку проєкту:
   ```bash
   cd logitrack-api
